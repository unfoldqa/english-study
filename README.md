# English Course A1 → B2

Полный курс английского в формате HTML: **64 урока** (16 на уровень), от A1 до B2. Программа выстроена по принципам Oxford/Cambridge communicative syllabus и рамкам CEFR.

## Онлайн

**https://unfoldqa.github.io/english-study/**

## Локально

```bash
open index.html
```

## Структура

| Файл | Описание |
|------|----------|
| `index.html` | Главная страница с прогрессом |
| `lesson.html?id=N` | Урок (N = 1…64) |
| `assets/course.js` | Логика уроков, проверка ответов, распознавание речи |
| `assets/curriculum.js` | Контент 64 уроков |
| `assets/enrich_curriculum.py` | Добавляет теорию и мини-тесты к урокам |

## Программа (64 урока)

- **A1** (1–16): приветствия, to be, вопросы, артикли, Present Simple, there is/are, can, семья, еда, погода, покупки, хобби, здоровье
- **A2** (17–32): времена, Present Perfect, Future, сравнения, passive, герундий, условные 1 типа, used to, путешествия
- **B1** (33–48): модальные, условные 2–3 типа, reported speech, relative clauses, phrasal verbs, wish, каузатив
- **B2** (49–64): словообразование, коллокации, регистр, эссе, инверсия, переговоры, презентации, экзамен B2

## Каждый урок включает

1. **Теория** — объяснение по CEFR с ключевыми пунктами
2. **Разогрев** — 3 вопроса с автопроверкой
3. **Лексика** — карточки + мини-тест
4. **Грамматика** — правила, примеры, проверка
5. **Тест** — 5 вопросов с выбором ответа
6. **Произношение** — IPA, прослушивание, проверка фразы (текст / голос)
7. **Карточки** — 20 слов с отметкой «знаю / ещё раз»
8. **Культура** — материалы + вопросы с проверкой
9. **Говорение** — устные задания с микрофоном (Web Speech API) и проверкой
10. **Завершение** — доступно после прохождения всех блоков

## Обновление контента

```bash

python3 assets/enrich_curriculum.py # добавить теорию и vocabQuiz
```
