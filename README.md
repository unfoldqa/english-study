# English Course A1 → B2

Курс английского в формате HTML: **64 модуля**, **~192 часа** (в диапазоне 150–200 ч интенсивного обучения с домашкой).

## Онлайн

**https://unfoldqa.github.io/english-study/**

## Реалистичный объём (CEFR)

| Компонент | Часы |
|-----------|------|
| Интерактив на платформе | ~43 ч (64 × 40 мин) |
| Домашняя работа | ~149 ч (чтение, аудио, письмо, SRS, речь) |
| **Итого A1→B2** | **~192 ч** |

В академической программе (Cambridge/Oxford) тот же путь — 500–600 ч. Наш формат — интенсив: меньше часов в классе, больше самостоятельной практики.

**Темп:** 5 ч/нед → ~38 недель (~9 месяцев).

## Структура одного модуля (~3 ч)

1. **Сессия 1 (40 мин)** — теория, лексика, грамматика, тесты, произношение, говорение на сайте
2. **Сессия 2 (60 мин)** — чтение и аудирование из блока «Культура»
3. **Сессия 3 (80 мин)** — письмо, карточки Anki, 15 мин устной практики без экрана

## Программа

- **A1** (1–16): базовое общение, семья, еда, покупки…
- **A2** (17–32): времена, passive, условные, путешествия…
- **B1** (33–48): модальные, reported speech, phrasal verbs…
- **B2** (49–64): эссе, регистр, переговоры, экзамен B2…
